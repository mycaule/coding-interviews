
Trouver les k tels que 3^k = 2^k mod 37

- Par démonstration:
```
    3^k = 2^k [37]
<=> 3^k x 19^k = 2^k x 19^k [37]
<=> (3 x 19)^k = 38^k [37]
<=> 57^k = 1 [37]    car 38 = 1 [37]
<=> 20^k = 1 [37]    car 57 = 20 [37]
<=> 20^k - 1 = 0 [37]
<=> k = 36 (petit théorème de Fermat)
```
